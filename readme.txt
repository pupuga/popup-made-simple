== Description ==
The Popup made simple is a WordPress plugin that enables you to easily add customizable pupup to your pages using the Gutenberg editor. 

== Features ==
You must add class "accept-action" to button if you want to use custom "Accept" button

== Installation ==
Go to your WordPress dashboard. Install the plugin either via the WordPress plugin directory or by navigating to Plugins > Add New > Upload Plugin. Choose the downloaded zip file and click Install Now. Then, activate the plugin.
